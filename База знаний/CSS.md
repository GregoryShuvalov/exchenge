**Как работать с псевдоклассами и псевдоэлементами**

Используйте псевдоклассы, чтобы применить стили к элементам в определенных контекстах или состояниях. Например, задайте стиль кнопки для ситуации, когда пользователь наводит на нее курсор.

| :hover         | применяет стили к элементу, когда пользователь наводит на него курсор;                                                                                   |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| :active        | применяет стили к элементу, когда пользователь кликает по нему клавишей мыши;                                                                            |
| :focus         | применяет стили к элементу, на котором установлен фокус;                                                                                                 |
| :first-child   | применяет стили к первому дочернему элементу внутри родительского элемента;                                                                              |
| :nth-child (n) | применяет стили к элементу, порядковый номер которого внутри родительского элемента равен n (например, :nth-child (3) выбирает третий дочерний элемент). |
|                |                                                                                                                                                          |
Примеры:

| Чтобы кнопка при наведении на нее курсором стала синей, <br> а текст белым, используйте следующий код.<br>  Между селектором и псевдоклассом пробел не ставится. | button:hover{<br>	background-color: blue;<br>	color: white;<br>} | <!DOCTYPE HTML><br><html><br><head><br><style><br>    button:hover{<br>    background-color: blue;<br>    color: white;<br>}<br></style><br></head><br><body><br>    <button>Наведите курсор на текст</button> <br> </body> |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

Так работают основные псевдоэлементы:

| ::before       | вставляет контент перед содержимым элемента;                                                                                                             |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ::after        | вставляет контент после содержимого элемента;                                                                                                            |
| ::first-line   | применяет стили к первой строке текста внутри элемента;                                                                                                  |
| ::first-letter | применяет стили к первой букве текста внутри элемента.                                                                                                   |
|                |                                                                                                                                                          |
Чтобы поставить стрелочку «->» перед содержимым элемента с классом .arrow, используйте псевдоэлемент ::before следующим образом:

.arrow::before{
	content: "->";
}


Блочная модель:
<!DOCTYPE html>
<html>
<head>
<style>
    .size {
    width: 50%; /*Элемент эелемент займет половину ширины экрана*/
    height: 100px; /*Высота элемента 100 пикселей*/
    background-color: rgb(114, 208, 206);
    font-size: 20px;
    text-align: center;
    font-family: Gilroy;
    float: right;
    }
</style>
</head>
<body>
    <p class="size">Текст</p>
</body>
</html>
